# Quintas Europeias

Quintas Europeias is a web application to track weekly football matches among friends.  
It allows you to:

- Register matches and teams
- Track player statistics: goals, wins, draws, losses, participation %
- Display leaderboards and top scorers
- View overall and individual player performance

Built with Django (backend) and React with Vite, TypeScript, Tailwind CSS, and shadcn/ui (frontend).
